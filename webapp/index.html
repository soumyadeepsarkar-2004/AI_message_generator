❓ Help & User Guide

🚀 Getting Started

  1. Enter a natural language prompt describing your message
  2. Click "Generate Message" to get AI-powered suggestions
  3. Customize placeholders with your specific details
  4. Copy or download your final message

💡 Example Prompts

  • "I want to send Diwali wishes to my customers"
  • "Create a Christmas greeting for clients"
  • "Welcome message for new users"
  • "Promotional offer announcement"
  • "Appointment reminder for patients"

🎯 Tips for Better Results

  • Be specific about the occasion or purpose
  • Mention your target audience (customers, clients, etc.)
  • Include context like industry or business type
  • Use keywords related to the message type

🔧 Features

  • Smart Matching: AI automatically detects message type
  • Alternatives: Multiple message variations to choose from
  • Customization: Easy placeholder replacement
  • History: Keep track of generated messages
  • Analytics: View usage statistics and trends
